【填空题】：写出下列反应的化学方程式

(1)黑火药是我国古代四大发明之一，它是由硝石（\(\ce{ KNO3}\)）、木炭、硫粉组成，点燃后产生比原来体积大千 倍的气体，具有猛烈的爆炸威力，反应后生成硫化钾、二氧化碳和一种气体单质，写出火药爆炸的化学方程式【答案：\(\ce{2KNO3 +3C +S \xlongequal {点燃} K2S +3CO2\uparrow +N2\uparrow}\)】.

(2)一氧化氮是大气的主要污【答案：\(\ce {4NH3 +5O2}\xlongequal[\bigtriangleup]{催化剂}\ce{4NO +6H2O} \)】染物之一，近几年【答案：\(\ce {4NH3 +5O2}\xlongequal[\bigtriangleup]{催化剂}\ce{4NO +6H2O} \)】发现生物体内存在少量一氧化氮，它有扩张血管、增强记忆的功能，成为当前生命科学【答案：\(\ce {4NH3 +5O2}\xlongequal[\bigtriangleup]{催化剂}\ce{4NO +6H2O} \)】研究的热点.工业上用氨气（\(\ce{ NH3}\))和氧气在有催化剂和【答案：\(\ce {4NH3 +5O2}\xlongequal[\bigtriangleup]{催化剂}\ce{4NO +6H2O} \)】加热的条件下反应可得到一氧化氮和另一种氧化物.写出该反应的化学方程式【答案：\(\ce {4NH3 +5O2}\xlongequal[\bigtriangleup]{催化剂}\ce{4NO +6H2O} \)】.
